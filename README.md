# moviedb

> Movie App with Vue.js
> This is a simple app made with themoviedb api and vuejs along with axios.
> It displays 10 top rated shows and movies, there is also a search bar for the movie database
> to look through and get details on certain show or movie you choose.

> When downloading the app make sure you have Node.js installed, and simply follow the instructions bellow.

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
